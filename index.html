what is HTML!

(HTML stands for Hypertext Markup Language)

it's the markup language that the web developer use to structure and describe the content of a webpage (NOT a programming language)

HTML consist of elements that describes the different types of content, paragraphs, headings, links, images, video E.T.C . web browsers understands HTML and render HTML code as websites.

in HTML, each elements is made up of 3 parts

  1. The Opening Tag
  2. The Closing Tag
  3. The Actual Element
